AI PDF Summarizer (Offline GUI Tool)

This is a simple offline PDF summarizer tool built using Python and Tkinter.
It allows you to upload a PDF file, choose summary precision, select an AI model, and generate a text summary instantly — all without internet or external APIs.

This project is designed as a demo application for students and beginners.

Features

• Upload any PDF via “Browse”
• Choose summary precision: Low, Medium, High
• Select an AI model (simulated):

Meta-Llama 38B Instruct

Mistral 7B Instruct

Gemma 7B Instruct
• View summary inside the app
• Automatically save the summary as a .txt file
• 100% offline (no GPU or internet needed)
• Lightweight and ideal for GitHub / college submissions

How to Download

Click the green “Code” button on GitHub →
Choose “Download ZIP” → Extract it.

How to Run the Application
1. Install Python

Download Python 3.10+ from the official website.
Make sure to tick Add Python to PATH during installation.

2. Install required library

Open Command Prompt in the project folder and run:
pip install PyPDF2

3. Run the app

python pdf_summarizer.py
The GUI window will open.

Usage Guide

• Click Browse to select any PDF file
• Choose summary precision: Low / Medium / High
• Choose an AI model (simulated)
• Click Summarize
• Summary appears in the text box
• A .txt file is auto-saved in the same folder as your PDF

Technologies Used

• Tkinter (GUI)
• PyPDF2 (PDF text extraction)
